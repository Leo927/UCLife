# Phasing

| Phase | Scope | Demo moment |
|---|---|---|
| **0. Skeleton** | Vite/React/Koota/Konva/Lingui set up; tile renderer; clock with pause+1×+2×+4×; player walks a hand-placed throwaway map | "I wander an empty district and the clock advances" |
| **1. Survival core** | Vitals; eat/sleep/wash interactables; death-from-neglect; commitment-skip for sleep | "I survive a week" |
| **2. Skills + economy** | Inventory; skill set; one job (work shift → wage); shops; books for skill grind | "Work → eat → train → sleep" loop closes |
| **3. NPC utility AI + procgen** | Procedural Von Braun district; 10–20 named NPCs with schedules/drives/pathfinding/memory; relations | "Sleep-skip and wake to a different station" |
| **4. Physiology** | Sickness, injury, contagion, clinic, permadeath toggle | "Flu spreads through a workplace" |
| **5.0 Ambitions** | Authored long-arc goals; player pursues any number from a menu of 6+; milestone payoffs (titles, log lines, dialogue unlocks, **Ambition Points**); perk catalog spendable from AP pool. `warPayoff` field present but inert. See [social/ambitions.md](social/ambitions.md). | "I have a target tomorrow morning, and the perks I earned last week make it cheaper" |
| **5.1 Newsfeed** | Date-keyed event content table; bar TV channel; journal HUD panel showing consumed headlines | "I duck into Lazlo's to catch the news" |
| **5.2 Talk verb + relationships** | "Talk" interaction on every NPC; per-NPC opinion; player-presence memory; ambitions involving NPC friendships start working | "The bartender remembers me" |
| **5.3 Factions deepen** | Federation + Zeon visible presence (consulates, uniformed NPCs); reputation deepens; faction-aligned ambitions earn real stakes | "My loyalty matters; war rumors land" |
| **5.4 Mobile worker subsystem** | First concrete verb behind the piloting skill: Sim Pod + MW Operator job + AE/Federation gated trials. Inertia-cockpit minigame. See [mobile-worker.md](mobile-worker.md). | "I bring a load through the gates and the cockpit feels different than it did yesterday" |
| **5.5 Civilian faction** | Facility ownership (Owner = Character \| Faction). Realtor per district sells existing-facility stock; private sales route through the embodied seller. Daily payroll + maintenance with 3-day insolvency and an embodied warning loop. Faction office + secretary delegate. HR office + recruiter NPC; applicants are procgen NPCs in the lobby. Bed pressure drives loyalty drift. Player-faction-creation dialogue migrates aliased fund/properties into a real entity. See [social/facilities-and-ownership.md](social/facilities-and-ownership.md). | "I buy the bar from its bankrupt owner, hire the regulars, and forget to make payroll" |
| **5.5.6 Research + facility tiers** | Research lab as `factionMisc` facility class. `FactionStatSheet` + `FactionEffects` (Effect engine reused for factions) + `FactionUnlocks: Set<string>` (binary gates kept off the stat sheet). `researchSystem` at `day:rollover`; queue + overflow + lost-if-empty with visible reporting. Researcher consultative talk-verb opens the planner panel. Universal four-knob facility-tier system (job-site count, efficiency, hours, loyalty drift) on every owned facility, surfaced on the worker's manage-interactable with secretary fallback. First research = `factory-tier-2` unlocks the visible-but-locked tier-2 row on the factory. See [social/research.md](social/research.md) and [social/facility-tiers.md](social/facility-tiers.md). | "I queue Factory Capacity II at Lazlo's neighbour-lab, then three weeks later walk into my own factory and the second-shift expansion is finally clickable" |
| **6.0 First ship + tactical foundation** | Player buys their first ship. Starsector-shape tactical view (top-down 2D, real-time + pause). Walkable current ship as scene, including a per-ship **captain's office** room adjacent to the bridge (pre-launch readiness summary read off ship state). Earth Sphere continuous-space campaign map opens (Sides 1–7, Luna, Luna II, Earth orbit, asteroid clusters) — see [starmap.md](starmap.md). **Combat event log** surface lands with the tactical foundation (Starsector-shape top-left, fading scroll, severity tiers); **tactical auto-pause set narrowed** to first-contact + flagship 25%/10% + boarders + player-eject — routine status changes route to the log. **Tally dialogue minimum** at post-combat (credits + supplies + fuel). Single-ship pre-war merc work. See [post-combat.md](post-combat.md). | "I burn from Granada to Side 6, dock at a derelict, and decide whether to fight or run" |
| **6.1 Bridge ↔ hangar walk + MS pilot** | Player walks the ship's rooms; physically transits bridge → hangar to climb into an MS. Cockpit primitives now have hostile counterparts. **MS launch + cockpit transition + dock-back** lands; single placeholder hangar door per ship (full `hangarDoors[]` authoring is 6.2.5). **No per-MS resources yet** — MS combat is hull-only; "run out of ammo" isn't a state. Walking-transit cost on ship AI (leaving bridge mid-combat = AI-piloted ship). Combat event log gets full content (launch / dock / kill / threshold entries + named-officer chatter). See [sortie.md](sortie.md). | "I leave the bridge mid-fight to take an MS personally — and feel my fleet wobble while I'm gone" |
| **6.1.5 Fleet structural prep** | Singleton-`Ship` → plural. Ship-template/instance split (`ship-classes.json5`); `IsFlagshipMark` marker. Save-handler migration to per-ship state. No new gameplay. See [fleet.md](fleet.md). | "Existing save loads with a fleet of one; nothing visible changed" |
| **6.2 Multi-ship fleet MVP (ships only, no retrofit)** | Two more ship classes (escort + small freighter). Per-ship + crew supply economics (Starsector model). Fleet roster + crew assignment screens. Hire-as-captain / hire-as-crew dialogue branches. Buy-ship dialog at brokers — purchase delivers a ship with its authored loadout, no fitting screen. **Comm panel relocates to the captain's office** (multi-ship needs it). **Officer-led crew auto-man** verb on the captain (talk → "man the rest from idle pool"). **`brigCapacity` stat + brig as ship-class room** (capacity gating only; per-prisoner verbs land at 6.2.5). **Tally dialogue full** (loot routing + named POW reveal); **notable-hostile authoring** on `space-entities.json5` rows (named captains in tactical, named in event log on death). Mothballing. Doctrine sliders. Persistent fleet damage. Debug "grant fleet" populates a 2-ship fleet + ~30 hired NPCs. Capacity is gated by economics + command bandwidth, not a skill formula. **Ships do not refit** — that depth lives in 6.2.5 on the MS. See [fleet.md](fleet.md), [post-combat.md](post-combat.md). | "My old bartending friend captains my second ship — and a captured pirate's portrait stares back at me from the post-combat tally" |
| **6.2.5 MS + pilot + retrofit + sortie depth** | The customization-depth phase. `ms-classes.json5`, `ms-weapons.json5`, `ms-frame-mods.json5`. MS runtime entity with per-instance `mountedWeapons` + `frameMods`. Hangar UI, pilot roster + assignment, per-MS supply + per-MS-repair supply economics. **MS retrofit panel** — weapon swap, frame-mod install, role tags. Buy-MS dialog at MS brokers. Auto-assign with manual override. **Per-MS sortie resources** (`currentPropellant`, `currentAmmoByWeapon`, `currentLifeSupport`); **mid-combat resupply protocol** (15s base, full crew/boost formula); **per-ship-class `hangarDoors[]` authoring + door queueing**; stranded-MS recovery tug. **Prisoner verbs** on the brig walk-up + captain's office comm panel (interrogate / ransom / recruit / execute / hand-over / release); in-flight prisoner upkeep via brig-condition stats. **MS-parts loot** via per-class salvage table. See [sortie.md](sortie.md), [post-combat.md](post-combat.md). | "I dock my dry GM mid-fight, the resupply timer ticks, I take the helm — fifteen seconds later the log says she's hot and I sprint back down the hall" |
| **6.2.7 CP / DP** | Command points + deployment points wired into tactical combat. Doctrine sliders fully active. | "I run out of CP coordinating four wings and they fall back on standing orders" |
| **6.3 Colony establishment + ship recovery** | Player can claim an asteroid base or build a new colony from scratch. Walkable colony scenes (smaller than Von Braun, industrial facility pool). Resupply hub, recruitment depth, income, pirate raids. Colony-only facility classes (warship slipway, large MS factory) gate capital production. Colony scale gated by Leadership administrative load (no hard cap). **Recoverables dialogue full** — capture / salvage / scuttle each surviving hull and pod; prize-crew gate. **Salvaged-hull-in-flight pattern** — recovered hull joins the fleet immediately, queues delivery on next dock like a fresh purchase. **`WasCaptured` marker + faction-relation hooks**; **colony detention** as brig-overflow target. See [social/faction-management.md](social/faction-management.md), [post-combat.md](post-combat.md). | "I capture a Musai with the prize crew I had warming the bench, fly her two days to my own colony, and watch her lower into a real slip" |
| **6.4 Faction tier** | Player-faction acquires its own faction reputation slot. Recruitment scale (open calls on colonies). Governance choices. Diplomacy with canon factions. Faction-leader perks unlock for the AP pool. Jupiter expedition ships as a frontier long-arc option. | "Federation patrols salute my flag, and an AE diplomat wants a meeting" |
| **7. War event** | UC 0079.01.03 hits; world state shift; factions go hostile; Von Braun changes; `warPayoff` routes resolve the most-progressed ambition. Civilian-war content (newsfeed wartime mode, conscription, refugees, departing friends), real cockpit combat for the pilot path, wartime faction management, hostile expeditions vs player colonies. See [combat.md](combat.md) for the full structural design. | The prologue payoff |
| **8. LLM dialogue + intent** | Persona-cached Claude prompts for dialogue and proposed actions | "Talking to NPCs feels alive" |

## Open / deferred

- Permadeath UX (unlock in Phase 4)
- Save migration policy when traits churn pre-1.0
- LLM cost model and rate-limiting (Phase 8 design)
- Sound design (deferred indefinitely; ambient tracks + UI clicks adequate)
- Mobile suit subsystem when UC 0079+ unlocks mobile suit access (uses the unified Piloting skill)
- Character creator UX (sets stat talents per [characters/attributes.md](characters/attributes.md)); NPC talent randomization via seeded RNG. Until both land, every character launches at talent = 1.0.
- Stat balance pass after first playtests of attributes: tune `DRIFT`, the floor, and the use/stress feed weights once the harshness reads in actual play.
