# Helper file origin

These files are extracted verbatim from FC pregmod (GPL-3.0-or-later):

| File | FC source |
|---|---|
| `eyeChecker.js` | `src/js/statsChecker/eyeChecker.js` (full file) |
| `limbChecker.js` | `src/js/statsChecker/limbChecker.js` (full file) |
| `burstChecker.js` | `src/js/statsChecker/burstChecker.js` (full file) |
| `pregChecker.js` | `src/js/statsChecker/pregChecker.js` (full file) |
| `artHelpers.js` | `src/art/artJS.js` lines 858–2900 (extractColor, clothing2artSuffix, skinColorCatcher) |

`statsChecker.js` was deliberately NOT copied — it has heavy `V.*` coupling to
arcology/menstruation/dairy systems that don't apply to UC, and the renderer
doesn't reference its functions.

The artJS.js extract preserves the verbatim helper text. The surrounding 96 KB
of artJS.js handles WebGL/AI/rendered-art paths and `<<SlaveArt>>` macros that
Phase A doesn't need; Phase D will re-attach the rest.
